## Purpose

The GM's Toolkit is the comprehensive GM resource, combining everything GMs need to run sessions, build narratives, create NPCs/VPCs/enemies, and manage campaigns. This book contains **GM-only content** with potential player spoilers and focuses exclusively on the GM's role.
