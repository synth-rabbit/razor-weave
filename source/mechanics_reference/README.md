## Purpose

The Mechanics & Rules Reference is the authoritative source for all game mechanics, providing universal rules that apply across all genres and play styles. This book serves both **players and GMs** as the definitive mechanical reference, kept separate from narrative/setting content for easy lookup.

## Standalone Philosophy

This book is designed to be **completely standalone**:

- Can be used without Player's Handbook or GM's Toolkit (though less context)
- Contains all mechanical rules needed to play
- No narrative/setting content (kept in other books)
- Universal across all genres
- Quick lookup optimized (comprehensive index, tables, quick refs)

**Use Cases**:
- Players referencing skills/proficiencies mid-game
- GMs checking DCs or combat rules
- Quick rule lookups during play
- Learning the mechanics system
- Porting mechanics to other settings/games
