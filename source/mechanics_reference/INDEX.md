## Standalone Usage

This book is designed to be **completely standalone**:

- **For Players**: All rules needed to create characters and play
- **For GMs**: All mechanics needed to run sessions and resolve actions
- **Universal**: Works across all genres (Cozy, Fantasy, Horror/Mystery, Modern, Sci-Fi)
- **No Dependencies**: Can be used without Player's Handbook or GM's Toolkit (though less context)

### Quick Lookup Optimized

- **Alphabetical glossaries**: Tags & Conditions (Chapter 18)
- **Genre-organized**: Skills and Proficiencies grouped by setting
- **Cross-genre conversion**: Examples show how to adapt between genres
- **Complete index**: This page for quick navigation

1. Choose genre (Chapters 7-11, 13-17)
2. Select 3-5 skills from genre chapter
3. Select 2-4 proficiencies from genre chapter
4. See Player's Handbook for full creation process

1. Quick lookup: Chapters 3, 18, 19
2. Skill checks: Chapter corresponding to your genre
3. Combat: Chapter 19
4. Conditions: Chapter 18

**Porting Between Genres**:
1. Identify character concept (core role/function)
2. Find equivalent skills in new genre chapter
3. Find equivalent proficiencies in new genre chapter
4. Swap gear for genre-appropriate items
