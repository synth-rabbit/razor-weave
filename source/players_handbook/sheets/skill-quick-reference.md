## Skill Check Modifiers

**Add +2 to DC** when:
- Time pressure or rushed
- Poor tools or conditions
- Active opposition
- Unfamiliar environment

**Subtract 2 from DC** when:
- Perfect conditions
- Ideal tools
- Expert assistance
- Prepared/practiced

**Grant Advantage** when:
- Preparation and planning
- Ally assists effectively
- Favorable circumstances
- Using relevant proficiency

**Impose Disadvantage** when:
- Improvising or rushed
- Injured or impaired
- Hostile environment
- Lacking necessary tools

## Combining Skills

**When task requires multiple skills**, use highest skill with advantage if both apply:

**Examples**:
- **Stealth + Acrobatics**: Silent parkour
- **Persuasion + Lore**: Teach convincingly
- **Investigation + Medicine**: Forensic analysis
- **Combat + Athletics**: Dynamic fighting

## Quick Skill Selection Guide

**Choose skills based on what you enjoy**:

**I want to fight**: Combat skills (Melee, Ranged, Tactics)
**I want to talk**: Social skills (Persuasion, Deception, Insight)
**I want to know things**: Knowledge skills (Lore, Investigation, specialty)
**I want to move**: Physical skills (Athletics, Acrobatics, Stealth)
**I want to make things**: Technical skills (Crafting, Engineering, specialty)
**I want versatility**: Mix of categories
